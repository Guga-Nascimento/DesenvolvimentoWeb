className e classList no DOM

Aula prática para entender como o JavaScript manipula classes CSS.

Título da seção

Este bloco começa com as classes container e container-dark.

Controles da aula

Clique em um botão para ver o resultado na tela.

Diferença importante

JavaScript é a linguagem que escreve a instrução. DOM é a estrutura da página que o navegador entrega ao JavaScript.

Então, className e classList não são “coisas do CSS”. Eles são recursos do elemento no DOM, usados pelo JavaScript para alterar as classes.

Funções e recursos usados no navegador

  • document.getElementById() → procura um elemento do HTML pelo id.
  • addEventListener() → faz o navegador esperar um evento, como clique.
  • textContent → altera o texto mostrado dentro do elemento.
  • JSON.stringify() → transforma array em texto para exibir melhor na tela.

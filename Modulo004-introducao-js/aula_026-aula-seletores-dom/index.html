Estudo de Seletores no DOM

Esta página mostra, na prática, como funcionam querySelector() e querySelectorAll().

O que está acontecendo aqui?

Os seletores usados no JavaScript seguem a mesma lógica dos seletores do CSS. Ou seja: o JS usa a linguagem de seleção, mas quem executa a busca é o navegador via DOM.

  • JavaScript: linguagem que escreve a instrução.
  • DOM/Navegador: estrutura que contém os elementos HTML e permite buscá-los.

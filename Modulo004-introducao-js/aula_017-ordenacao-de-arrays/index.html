Funções usadas no navegador

  • document.getElementById() pega um elemento do HTML pelo id.
  • addEventListener() faz o navegador esperar uma ação, como um clique.
  • textContent troca o texto que aparece na tela.
  • JSON.stringify() transforma array em texto para exibição.
  • sort() ordena os elementos do array.
  • reverse() inverte a ordem atual do array.

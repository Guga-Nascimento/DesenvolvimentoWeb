Diferença importante

JavaScript é a linguagem que escreve os comandos. DOM é a estrutura da página que o navegador entrega ao JavaScript.

Então, getAttribute(), setAttribute() e removeAttribute() são recursos do DOM usados pelo JavaScript para ler e alterar atributos de elementos HTML.

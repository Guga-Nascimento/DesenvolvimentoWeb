Aula prática para entender como um botão reage a um clique com JavaScript.

Controles da aula

O que está acontecendo aqui?

Quando você clica no botão, um evento acontece. O JavaScript escuta esse evento e executa uma função.

Neste exemplo, vamos usar addEventListener() para reagir ao clique.

Saída da explicação

Clique em um botão para ver a ação acontecendo na tela.

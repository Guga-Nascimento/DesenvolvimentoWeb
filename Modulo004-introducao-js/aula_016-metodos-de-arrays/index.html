Métodos de Arrays

Este exemplo mostra, na prática, como usar alguns métodos importantes de arrays no JavaScript.

Array original

["joao", "ana", "carlos"]

Resultado

Clique em um botão para testar um método do array.

O que o JavaScript usa aqui no navegador?

  • document.getElementById() → pega um elemento do HTML pelo id.
  • addEventListener() → fica “escutando” um evento, como clique no botão.
  • textContent → troca o texto que aparece dentro do elemento.
  • JSON.stringify() → transforma array ou objeto em texto para exibir na tela.

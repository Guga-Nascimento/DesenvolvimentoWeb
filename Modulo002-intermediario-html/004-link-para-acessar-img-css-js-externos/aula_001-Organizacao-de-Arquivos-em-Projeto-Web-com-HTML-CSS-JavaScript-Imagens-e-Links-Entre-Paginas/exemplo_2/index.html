Carregar ficheiros externos

Nesta página estamos a aprender como ligar um ficheiro CSS, um ficheiro JavaScript, uma imagem e também links para outras páginas.

Notícias
Serviços

O que está a acontecer aqui?

HTML: cria a estrutura da página.

CSS: altera o visual da página.

JavaScript: adiciona comportamento.

Imagem: é carregada pela tag img.

Links: abrem outras páginas da pasta outras_paginas.

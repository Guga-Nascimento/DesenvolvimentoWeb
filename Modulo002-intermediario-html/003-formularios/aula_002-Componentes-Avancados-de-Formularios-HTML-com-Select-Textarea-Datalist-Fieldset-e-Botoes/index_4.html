O que estamos fazendo aqui?

Estamos usando datalist para criar uma lista de sugestões ligada a um campo de texto.

No navegador, o usuário verá um campo para digitar. Ao clicar nele ou começar a escrever, aparecem sugestões como Portugal, Brasil e Angola.

Diferente do select, o usuário pode escolher uma sugestão ou digitar outro valor manualmente.

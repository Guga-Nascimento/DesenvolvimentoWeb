Formulário

O que estamos fazendo aqui?

Estamos criando uma lista de seleção com a tag select.

Dentro dela, usamos option para criar as opções que o usuário pode escolher.

Também usamos optgroup para separar as opções em grupos, deixando a lista mais organizada.

No navegador, o usuário verá uma caixa de seleção. Ao clicar nela, aparecerão os grupos e os países disponíveis para escolha.

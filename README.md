💻 Desenvolvimento Web

Repositório com minha trilha prática de estudos em desenvolvimento web, com foco na construção de projetos reais e evolução contínua até aplicações completas.

🚀 Sobre o projeto

Este repositório reúne minha jornada de aprendizado em desenvolvimento web, passando pelos fundamentos até tecnologias utilizadas no mercado.

O conteúdo está organizado por módulos, cada um focado em uma tecnologia ou etapa específica da construção de aplicações web.
